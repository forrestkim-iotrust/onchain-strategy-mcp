No copy-pasting Python. No exporting keys. No tool stack to cobble together.

Local-first

The runtime, the keys, the state — all on your machine. No servers we run.

Permissioned

Deny-by-default policy DSL gates every signing call. Outside the policy, nothing leaves the box.

Atomic by default

Multi-step plans bundle into one EIP-7702 transaction. No approve-and-pray gaps.

What you can build

AUTO-FUNNEL

Every drop of ETH or USDC that lands at your wallet ends up earning yield, automatically.

YIELD ROTATOR

Liquidity follows the best rate — only when the post-gas gain beats your threshold.

EVENT REACTOR

Oracle drops, transfer hooks, mempool signals — strategies fire onchain within seconds.

ATOMIC MULTI-STEP

approve + supply, swap + LP — land in one transaction via EIP-7702.

Install

Requires Node.js 18+ and Claude Code.

# Scaffold a burner (key → OS keychain), starter config + policy.

npx onchain-strategy-mcp init

# Register the MCP server with Claude Code.

claude mcp add osmcp -- npx onchain-strategy-mcp serve

Then open Claude Code and say “load the getting_started prompt and walk me through this MCP.” The server self-documents.

Build from source (advanced) 
git clone https://github.com/forrestkim-iotrust/onchain-strategy-mcp.git
cd onchain-strategy-mcp
cargo build --release --bin executor-mcp

cast wallet new
export EXECUTOR_PRIVATE_KEY=0xyourkey

cp -R .local.example .local
$EDITOR .local/config.toml
$EDITOR .local/policy.toml

claude mcp add osmcp \
  -e EXECUTOR_CONFIG=$PWD/.local/config.toml \
  -e EXECUTOR_PRIVATE_KEY=$EXECUTOR_PRIVATE_KEY \
  -- $PWD/target/release/executor-mcp

Requires Rust and (optionally) Foundry for cast.

FAQ

What chains does it support?

Built and tested on Base. Any EVM-compatible chain works with a one-line config change. Solana and other non-EVM chains are on the roadmap.

What does it cost?

The software is free. Onchain transactions cost gas — typically under $0.10 per action on Base. Start with $5–10 of ETH and a small amount of whatever asset you want to work with.

Do I need to be a programmer?

A little. You'll copy commands and edit a config once. Strategies are short JavaScript functions, which Claude will write from your plain-language description.

Do I need API keys?

For basic use, no — public RPC works. For mempool watching, live event subscriptions, or deep historical reads, you'll want an Alchemy key. Free tier is enough.

What is "MCP"?

Model Context Protocol — the standard way Claude Code (and similar AI clients) talk to outside programs. This project is one of those programs.

How does the policy model work?

Deny-by-default DSL — allowed chains, contracts, function selectors, native value caps, ERC20 spend caps. Anything outside the policy is refused before signing.

Where do I report bugs?

Open an issue on GitHub. Use the private security advisory feature for anything sensitive.
