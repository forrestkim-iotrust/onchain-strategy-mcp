시나리오

실제로 굴리는 것들.

A. 자동 입금 깔때기

내 burner에 ETH나 USDC 들어오면 자동으로 USDC로 바꿔서 Aave에 예치해줘. 가스용 ETH는 $0.10어치만 남기고.

Claude가 strategy와 log 트리거 두 개를 등록합니다. 지갑에 들어오는 모든 자금이 이자를 받기 시작해요. 신경 쓸 건 주소로 돈 보내는 거 하나.

B. 수익률 비교기

지난 30일 Aave / Compound / Moonwell USDC 이자율 시간 단위로 비교해줘.

Claude가 짧은 view 함수를 짜서 archive RPC로 과거 블록을 훑어 표로 즉시 돌려줍니다. 데이터는 이미 온체인에 있어요.

C. 즉각 반응자

Aave 오라클이 ETH 가격 업데이트하는 걸 봐. 한 번에 2% 이상 떨어지면 내 borrow 갚아.

Claude가 로그 이벤트 트리거를 등록하면, 이벤트가 발화하는 순간 strategy가 초 단위로 돕니다.

D. 원자적 멀티스텝

내 burner USDC 0.1을 Aave에 예치해줘.

Claude가 [approve, supply] 두 action을 반환하면 런타임이 EIP-7702 한 트랜잭션으로 묶어 broadcast합니다. 둘 다 들어가거나 둘 다 안 들어가요.

설치

명령어 두 줄. 1분 미만.

# 1. 한 줄 설치 (prebuilt 바이너리 다운로드 + burner 지갑 OS 키체인에 생성
#    + .local/config.toml + .local/policy.toml 스캐폴드)
$ npx onchain-strategy-mcp init

# 2. Claude Code에 등록 (init이 이 줄을 그대로 출력해줍니다)
$ claude mcp add osmcp -- npx onchain-strategy-mcp serve

Claude Code 열고:

examples/strategies/yield-snapshot.js 등록해서 한 번 돌리고 결과 보여줘.
소스에서 직접 빌드 (고급)

prebuilt 바이너리 대신 Rust 소스에서 직접 빌드하고 싶다면:

$ git clone https://github.com/forrestkim-iotrust/onchain-strategy-mcp.git
$ cd onchain-strategy-mcp
$ cargo build --release --bin executor-mcp

$ cast wallet new                       # burner 지갑 생성, 소액만
$ export EXECUTOR_PRIVATE_KEY=0x여기에키

$ cp -R .local.example .local
$ $EDITOR .local/config.toml            # RPC와 signer env 변수명
$ $EDITOR .local/policy.toml            # 에이전트 권한

$ claude mcp add osmcp \
    -e EXECUTOR_CONFIG=$PWD/.local/config.toml \
    -e EXECUTOR_PRIVATE_KEY=$EXECUTOR_PRIVATE_KEY \
    -- $PWD/target/release/executor-mcp

Rust 필요. cast 쓰려면 Foundry도.

FAQ

짧은 답변.

어떤 체인?

Base에서 빌드·테스트. EVM 호환 체인이면 config 수정으로 다 됩니다. Non-EVM은 로드맵에 있어요.

비용은?

소프트웨어는 무료. Base 가스비는 액션당 보통 $0.10 미만. 지갑에 $5~10어치 ETH와 운영할 자산 소액으로 시작하세요.

프로그래밍 필요?

조금. 명령어 복붙 + config 한 번 편집. Strategy 본체는 짧은 JS — Claude가 설명만 들으면 짜줍니다.

API 키?

기본 사용은 공개 RPC로 OK. mempool 감시·실시간 이벤트·과거 데이터는 Alchemy 키 권장. 무료 tier 충분.

"MCP"가 뭐?

Model Context Protocol — Claude Code가 외부 프로그램이랑 대화하는 표준. 이 프로젝트가 그 외부 프로그램 중 하나.

정책·권한 모델은?

deny-by-default DSL: 허용 체인, 컨트랙트, 함수 셀렉터, native value 한도, ERC20 지출 한도. 정책 밖은 서명 전 거부.

버그·문의?

issue 등록. 보안은 GitHub private security advisory 사용.
