AI 에이전트에게 온체인 손발을 달아줍니다.

Claude의 온체인 아이디어를 실제로 굴리는 로컬 런타임 — 로컬 서명, 브로드캐스트, 영수증 대기, 이벤트 반응, 그리고 멀티스텝 액션을 EIP-7702 한 트랜잭션으로 묶기까지.

이제 AI가 코드를 제안만 하는 게 아니라, 직접 굴립니다.

지금 Claude한테 "내 USDC를 Aave에 예치해줘" 하면? Python 코드 한 덩어리 던져주고 "이거 돌려보세요" 합니다. 그 뒤로는 다 사용자 몫이에요 — RPC, 키, 가스, 시뮬레이션, 영수증. 이 런타임이 그 운영자 자리를 컴퓨터한테 맡깁니다.

01

연결·시뮬레이션·서명·브로드캐스트

Strategy는 샌드박스 JS 런타임에서 실행됩니다. 체인 연결, 단계별 시뮬레이션, 로컬 서명, 브로드캐스트, 영수증 대기까지 자동.

02

모든 결정을 저널링

source 읽기, action, 로그가 로컬 SQLite 저널에 그대로 남습니다. 재현 가능, 감사 가능, 전부 사용자 컴퓨터 안.

03

온체인 이벤트에 즉시 반응

log, mempool, interval, manual 트리거를 붙이면 매칭 이벤트가 발화하는 순간 strategy가 초 단위로 돕니다.

04

원자적 멀티스텝 (EIP-7702)

[approve, supply]를 반환하면 런타임이 한 트랜잭션에 묶어 broadcast합니다. 둘 다 들어가거나 둘 다 안 들어가요.

기본 제공 트리거 모드
모드언제 발화하는가
manual사용자(또는 Claude)가 시킬 때
intervalN ms마다 — cron 스타일
log확정된 블록에서 매칭 로그가 나올 때
mempoolmempool에 매칭 pending tx (Alchemy WSS)
예약block, webhook — v1.3에서 연결
뭘 만들 수 있어요

실제로 굴려본 시나리오들.

A. 자동 입금 깔때기

내 burner에 ETH나 USDC 들어오면 자동으로 USDC로 바꿔서 Aave에 예치해줘. 가스용 ETH는 $0.10어치만 남기고.

Claude가 strategy와 log 트리거 두 개를 등록하면 깔때기가 알아서 돌아갑니다. 지갑에 들어오는 모든 자금이 이자를 받기 시작해요. 신경 쓸 건 주소로 돈 보내는 거 하나.

B. 수익률 비교기

지난 30일 Aave / Compound / Moonwell USDC 이자율 시간 단위로 비교해줘.

Claude가 짧은 view 함수를 짜서 archive RPC로 과거 블록을 훑어 표로 즉시 줍니다. 기다림 없음 — 데이터는 이미 온체인에 있어요.

이제부터 계속 보다가 Moonwell이 5% 넘으면 알려줘.

같은 런타임의 두 모드: 과거는 호출 한 번, 실시간 감시는 트리거.

C. 즉각 반응자

Aave 오라클이 ETH 가격 업데이트하는 걸 봐. 한 번에 2% 이상 떨어지면 내 borrow 갚아.

Claude가 로그 이벤트 트리거를 등록해두면, 그 정확한 이벤트가 발화하는 순간 strategy가 초 단위로 돕니다.

D. 원자적 멀티스텝

내 burner USDC 0.1을 Aave에 예치해줘.

Claude가 [approve, supply] 두 action을 반환하면, 런타임이 멀티스텝임을 감지해 EIP-7702 한 트랜잭션으로 묶어 broadcast합니다. approve와 사용 사이의 위험 구간이 사라집니다.

동작 방식

하나의 대화 루프. 네 개의 부품.

프롬프트Claude Code에서
→
Claude계획·도구 호출
→
MCP 런타임시뮬·서명·저널
→
정책 게이트deny-by-default
→
온체인broadcast·영수증

Strategy는 Action[] (또는 "noop")을 반환하는 짧은 자바스크립트 함수입니다. 정책 DSL에 없는 건 — 잘못된 체인, 미등록 컨트랙트, 한도 초과 — 서명 전에 거부됩니다.

설치

명령어 두 줄. 1분 미만.

준비물: Mac 또는 Linux, Node.js 18+, Claude Code. Rust도 Foundry도 필요 없음. 

# 1. 한 줄 설치 (prebuilt 바이너리 다운로드 + burner 지갑 OS 키체인에 생성
#    + .local/config.toml + .local/policy.toml 스캐폴드까지)
$ npx onchain-strategy-mcp init

# 2. Claude Code에 붙이기 (init이 이 명령어를 그대로 출력해줍니다)
$ claude mcp add osmcp -- npx onchain-strategy-mcp serve

끝. Claude Code 열고:

examples/strategies/yield-snapshot.js 등록해서 한 번 돌리고 결과 보여줘.

Claude가 MCP 도구로 strategy 등록·실행하고, journal에 남은 결과가 채팅에 그대로 나옵니다. 여기서부터는 strategy 더 짜고, 트리거 붙이고, 흐름을 말로 다 조립하면 됩니다.

소스에서 직접 빌드 (고급)

prebuilt 바이너리 대신 Rust 소스에서 직접 빌드하고 싶다면:

$ git clone https://github.com/forrestkim-iotrust/onchain-strategy-mcp.git
$ cd onchain-strategy-mcp
$ cargo build --release --bin executor-mcp

$ cast wallet new                       # burner 지갑 생성, 소액만
$ export EXECUTOR_PRIVATE_KEY=0x여기에키

$ cp -R .local.example .local
$ $EDITOR .local/config.toml            # RPC와 signer env 변수명
$ $EDITOR .local/policy.toml            # 에이전트 권한

$ claude mcp add osmcp \
    -e EXECUTOR_CONFIG=$PWD/.local/config.toml \
    -e EXECUTOR_PRIVATE_KEY=$EXECUTOR_PRIVATE_KEY \
    -- $PWD/target/release/executor-mcp

Rust 필요. cast 쓰려면 Foundry도.

FAQ

짧은 답변.

어떤 체인에서 돼요?

Base(L2)에서 빌드·테스트했습니다. EVM 호환 체인이면 config 한두 줄 바꾸는 걸로 다 됩니다 — 이더리움 메인넷, Arbitrum, Optimism, Polygon 등. Solana나 다른 non-EVM은 로드맵에 있어요.

비용은 얼마나 들어요?

소프트웨어는 무료. 온체인 트랜잭션은 가스비가 듭니다. Base는 보통 액션당 $0.10 미만. 더 큰 비용은 지갑 자체 — $5~10어치 ETH랑 운영할 자산 소액으로 시작하세요.

프로그래밍 할 줄 알아야 해요?

조금. 명령어 복붙이랑 config 파일 한 번 편집할 정도. Strategy 본체는 짧은 자바스크립트인데, Claude한테 뭐 하고 싶은지 말로 설명하면 알아서 짜줍니다.

API 키 필요해요?

기본 사용은 필요 없음. 공개 RPC로 OK. mempool 감시, 실시간 이벤트, 며칠 넘는 과거 데이터 조회는 Alchemy 키가 있으면 좋아요. 무료 tier로 취미용 충분.

"MCP"가 뭐예요?

Model Context Protocol — Claude Code(랑 비슷한 AI 클라이언트)가 외부 프로그램이랑 대화하는 표준 방식. 이 프로젝트가 그런 외부 프로그램 중 하나고, "Claude Code에 추가한다"는 건 "Claude야, 이 친구랑 얘기할 수 있어"라고 알려주는 거예요.

권한·정책 모델은 어떻게 동작해요?

런타임은 deny-by-default 정책 DSL이 기본으로 들어 있어요 (허용 체인, 컨트랙트, 함수 셀렉터, native value 한도, ERC20 지출 한도). 정책에 없는 건 서명 전에 거부됩니다. 단순한 baseline이고, 더 정교한 권한 분리(세션 키, 에이전트 전용 지갑)는 로드맵에 있어요.

버그 있거나 궁금한 게 있으면요?

issue 등록. 보안 관련은 GitHub private security advisory 기능 쓰세요.

로드맵

앞으로 만들 것.

  • 출시 · v1.3 Claude Code 한 줄 설치. npx onchain-strategy-mcp init 한 번으로 burner 지갑(OS 키체인) + config + policy 스캐폴드. CREATE2 deterministic BatchExec 주소라 7702 batching이 어느 체인에서든 작동.
  • 예정 Self-documenting MCP 세션. 서버의 instructions, prompts, resource template를 풍부하게 채워서 — 예시·트리거 패턴·action shape·흔한 함정까지 — 새 Claude Code 세션이 처음부터 뭘 할 수 있는지 자기가 압니다.
  • 예정 외부 오라클 트리거. Chainlink / Pyth / Redstone 가격 업데이트, 오프체인 피드, HTTPS 웹훅으로 strategy 발화. 에이전트가 현실 세계 신호에도 반응할 수 있게.
  • 예정 WaaS 통합. Privy, Turnkey, Coinbase MPC 같은 에이전트 전용 지갑을 1급으로 연결. 권한 분리를 지갑 레이어가 책임지게 — 팀·프로덕션·멀티테넌트용.
