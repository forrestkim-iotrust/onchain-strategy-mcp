## Cross-Phase Integration

End-to-end flow exercised by `verification_examples` (3 passed) and `verification_safety` (2 passed):

```
strategy_register
  → strategy_run
  → sandboxed JS returns Action[]
  → action validation
  → EVM simulation (anvil-backed)
  → policy check (deny-by-default)
  → local hot-wallet signing (env-var-only key)
  → broadcast (configured RPC)
  → receipt wait (timeout-bounded)
  → execution_get / execution://{run_id} resource (shared build_execution_report)
  → journal:// resources for source reads, validation, simulation, policy, action outcomes
```
